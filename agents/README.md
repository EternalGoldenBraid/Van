# Agents

This directory contains handoff notes and working memory for agents helping with
the van build.

Use it for context that is useful to an agent but does not belong in the main
build plan yet:

- current decisions and assumptions
- open questions
- recent reasoning that should not be lost
- boundaries between planning, shopping, and implementation

Keep these files concise. The build plan and scripts should remain the source of
truth once a decision is settled.
